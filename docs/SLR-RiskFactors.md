## 1. PICO Framework

| Elemen | Rincian |
|--------|---------|
| **P** | Dewasa umum yang menggunakan alat monitoring tidur |
| **I** | Parameter tidur (durasi, fragmentasi, postur, restlessness) + faktor risiko demografis/klinis |
| **C** | Model risiko CVD standar: Framingham, ACC/AHA ASCVD, WHO CVD Risk |
| **O** | Kejadian kardiovaskular mayor (MACE): infark miokard, stroke, gagal jantung, kematian CVD |

## 3. Jalur Patofisiologi (Evidence-Based)

```
Gangguan Tidur
      │
      ├─→ Aktivasi Sistem Saraf Simpatis ──→ Hipertensi nokturnal → Hipertrofi ventrikel kiri
      │
      ├─→ Disregulasi HPA Axis ──────────→ Kortisol↑ → Resistensi insulin → DM Tipe 2
      │
      ├─→ Inflamasi Sistemik (CRP↑, IL-6↑)→ Aterosklerosis → Stroke/MI
      │
      ├─→ Hipoksia Nokturnal (via OSA) ──→ Disfungsi endotel → Trombosis
      │
      └─→ Metabolic Dysregulation ───────→ Obesitas → Dyslipidemia → CVD
```

## 5. ZLEEP-CVD Score (ZCS) — Model Konseptual

**Total Skor: 0–100 poin**

| Range | Level | 10-Year CVD Risk | Rekomendasi |
|-------|-------|-----------------|-------------|
| 0–25 | Rendah | <5% | Maintenance, review 6 bulan |
| 26–50 | Sedang | 5–10% | Tingkatkan sleep hygiene, konsultasi dokter |
| 51–75 | Tinggi | 10–20% | Rujuk kardiolog, CPAP screening |
| 76–100 | Sangat Tinggi | >20% | Segera konsultasi spesialis |

### Modifier Sinergistik (dari [13, 17])
```javascript
// HTN + DM + Sleep Efficiency buruk → +10 poin
if (htScore >= 5 && dmScore >= 5 && seScore >= 5) ZCS += 10;

// BMI ≥30 + Supine dominan + Usia ≥45 → +8 poin (OSA proxy)  
if (bmiScore >= 7 && supineScore >= 3 && ageScore >= 6) ZCS += 8;

ZCS = Math.min(ZCS, 100); // cap at 100
```

## 6. Pemetaan Input: Manual vs IMU Otomatis

```
INPUT MANUAL (Profil Pengguna)      OTOMATIS dari IMU Sensor
─────────────────────────────       ──────────────────────────────
• Usia (tanggal lahir)              • Total Sleep Time (TST)
• Jenis Kelamin                     • Wake After Sleep Onset (WASO)
• Berat Badan (kg)                  • Sleep Efficiency (%)
• Tinggi Badan (cm) → BMI           • Sleep Fragmentation Index
• Riwayat Hipertensi (ya/tidak)     • Nocturnal Restlessness Score
• Nilai BP terakhir (opsional)      • Posisi Supine/Lateral/Prone
• Riwayat Diabetes (ya/tidak)       • Sleep Onset Latency (SOL)
• Nilai HbA1c (opsional)            • Sleep Schedule Consistency (SD)
• Status Merokok                    • Movement Episodes per hour
• Estimasi aktivitas fisik/minggu   • Position Change Frequency
```

## 7. Gap Penelitian

| # | Gap | Implikasi untuk ZLEEP |
|---|-----|----------------------|
| G1 | Tidak ada model CVD yang mengintegrasikan data IMU tidur real-time | ZCS mengisi gap ini |
| G2 | Semua model (FRS, PCE, SCORE2) berbasis lab | ZCS adalah model lab-free berbasis sleep + profil |
| G3 | Tidak ada tool yang menggabungkan postur tidur + gerakan + profil dalam satu scoring | IMU ZLEEP mengukur fragmentation, postur, restlessness otomatis |
| G4 | Model yang ada tidak memodelkan interaksi faktor sinergistik | ZCS menggunakan multiplicative modifier |
| G5 | Tidak ada tool monitoring CVD berbasis tidur untuk populasi Asia Tenggara | ZLEEP pionir untuk konteks Indonesia/ASEAN |
